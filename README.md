# 🎨 Etch-a-Sketch Web App

A simple interactive drawing application built with **HTML5 Canvas**, **JavaScript**, and **CSS**.  
Users can draw by clicking or dragging on the canvas, choose brush colors and sizes, change the background, and reset or reload the sketch.


---
## ✨ Features
- **Draw with mouse clicks or drags** on the canvas.
- **Color picker**: select brush color from a dropdown.
- **Brush size selector**: choose different circle sizes (2px – 30px). 
- **Background color selector**: switch between black or white canvas backgrounds.
- **Erase button**: clears the canvas while keeping the background color.
- **Reload button**: refreshes the entire page to reset everything.

---
## 🚀 Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Start drawing on the canvas using your mouse!

---

## 🖱️ Controls
- **Choose a Color** → Sets the brush color.
- **Select Size of Painter** → Adjusts the brush size.
- **Background Color** → Changes canvas background.
- **Erase** → Clears the canvas.
- **Reload** → Reloads the page.
